Desarrollo a Medida

Desarrollamos software a medida para tu negocio, empresa o marca personal.

  • Desarrollo de aplicaciones web y moviles a medida, bajo la metodología Scrum.
  • Usamos tecnologías de vanguardía, principales en entornos de programación de software libre.
  • Participas de manera activa en el desarrollo de tu sistema.
  • Soporte técnico y mantenimiento al sistema incluido por 3 meses.
  • Nuestra misión es desarrollar software de alta calidad y de facil uso.

Web Dinámica Básica

Sitios Web que muestran contenido que cambia diariamente y requieren de una base de datos básica.

  • Adaptable a multiples dispositivos
  • Base de datos a convenir: Relacional(MySQL) o No Relacional(MongoDB)
